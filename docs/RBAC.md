# RBAC Matrix

| العملية | Admin | Manager | Officer | Viewer |
|---|---|---|---|---|
| عرض القساميات | ✅ | ✅ | ✅ | ✅ |
| إضافة قسمة | ✅ | ✅ | ✅ | ❌ |
| تعديل قسمة | ✅ | ✅ | ✅ | ❌ |
| حذف قسمة | ✅ | ✅ | ❌ | ❌ |
| إدارة المستخدمين | ✅ | ضمن الإدارة | ❌ | ❌ |
| طباعة/تقارير | ✅ | ✅ | ✅ | ✅ |

## قيود النطاق الإداري
- `Admin`: يمكنه الوصول لكل الإدارات.
- بقية الأدوار: ترى فقط السجلات المرتبطة بـ`department` الخاصة بالمستخدم.
- كل استعلام API يمر عبر `DepartmentScopedMixin` لمنع تسرب البيانات بين الإدارات.
